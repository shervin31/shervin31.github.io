Graph ML · Fraud Detection

NetTrace

A graph-based fraud detection system that models accounts as nodes and transactions as edges. Combines Graph Neural Networks (GraphSAGE), Isolation Forest anomaly detection, and Louvain community clustering to surface coordinated fraud that transaction-level models miss.

ML Pipeline · Marketing Analytics

Customer Conversion Predictor

End-to-end ML pipeline that predicts term deposit subscriptions for a Portuguese banking dataset. Benchmarked 8 classifiers, tuned XGBoost with stratified cross-validation, and packaged preprocessing + model into a single deployable joblib pipeline.

03
LLMs · RAG

RAG-Powered Chatbot

A Retrieval-Augmented Generation demo that lets users query documents from URLs or uploaded PDFs/TXTs. Splits documents with recursive chunking, embeds with HuggingFace sentence-transformers, retrieves with an in-memory vector store, and answers using GPT-4o-mini — with the retrieved context shown alongside every answer.
